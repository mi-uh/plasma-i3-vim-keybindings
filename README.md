# i3plasma

KDE Plasma shortcuts inspired by i3 and Vim. `Meta` is the primary modifier, with navigation based on `H/J/K/L`.

## Key bindings

| Shortcut | Action |
|---|---|
| `Meta + H/L` | Focus the window to the left/right |
| `Meta + J/K` | Switch virtual desktop |
| `Meta + 1–5` | Switch to virtual desktop 1–5 |
| `Meta + Alt + 1–5` | Move the current window to virtual desktop 1–5 |
| `Meta + Alt + H/L` | Tile the current window left/right |
| `Meta + Shift + H/L` | Move the current window to the adjacent screen |
| `Meta + Shift + 1–3` | Move the current window to screen 0–2 |
| `Meta + Return` | Open Konsole |
| `Meta + F` | Open a Firefox window |
| `Meta + Alt + F` | Open a private Firefox window |
| `Meta + Alt + C` | Close the current window |
| `Meta + W` | Open the overview |

On German keyboard layouts, KDE displays `Meta + Shift + 1–3` as `Meta + !`, `Meta + "` and `Meta + §`.

## Installation

Open **System Settings → Keyboard → Shortcuts**, then import [`i3plasma-custom.kksrc`](./i3plasma-custom.kksrc).
